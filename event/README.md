| IE   | CH   | FF   | SF   | OP   | IOS  | Android | Node |
| ---- | ---- | ---- | ---- | ---- | ---- | ------- | ---- |
| 8+   | 29+  |      |      |      |      |         | 4+   |

**注意：编译代码依赖ES5环境，对于ie6-8需要引入[es5-shim](http://github.com/es-shims/es5-shim/)才可以兼容，可以查看[demo/demo-global.html](./demo/demo-global.html)中的例子**
